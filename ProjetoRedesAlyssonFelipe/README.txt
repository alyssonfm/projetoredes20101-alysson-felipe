Projeto de Redes de Computadores 2010-1
Desenvolvido por Alysson Filgueira e Felipe Barbosa.

Instruções:

Uso do projeto no Windows:

	Caso esteja usando o sistema operacional Windows, deve-se executar o ProjetoRedes_Alysson_Felipe.bat, o qual se encontra na raiz do diretório. No console escolha no menu se deseja alterar as configurações ou usar as configurações padrão e dê ENTER. Para matar um nó, dê Ctrl+c ou simplesmente feche a janela do console.


Uso do projeto no Linux:

	Caso esteja usando o sistema operacional Linux (Ubuntu por exemplo), deve-se mandar executar em terminal o arquivo ProjetoRedes_Alysson_Felipe.sh, que se encontra na raiz do diretório. No terminal que se abrirá, realize a escolha no menu e dê ENTER. Para matar um nó, dê Ctrl+c ou feche a janela do terminal.
