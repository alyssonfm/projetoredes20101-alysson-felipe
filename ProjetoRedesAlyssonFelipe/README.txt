Projeto de Redes de Computadores 2010-1
Desenvolvido por Alysson Filgueira e Felipe Barbosa.

Instruções:

Uso do projeto no Windows:

	Caso esteja usando o sistema operacional Windows, deve-se executar o ProjetoRedes_Alysson_Felipe.bat, o qual se encontra na raiz do diretório. No console escolha no menu entre:
	
	1 se deseja alterar as configurações; 
	2 usar as configurações padrão;
	3 para encerrar o programa; 
	e dê ENTER.
	
	Se escolher a opção 1, alterar as configurações, irá alterar o time-out dos roteadores, bem como o tempo para envio de cada pacote de dados.
	Após alterar as configurações você escolherá o roteador que deseja criar e iniciará o programa de roteamento. 
	
	Escolhendo a opção 2, irá escolher um roteador e usará as configurações definidas como padrão no programa.
	
	Para matar um nó, dê Ctrl+c ou simplesmente feche a janela do console.


Uso do projeto no Linux:

	Caso esteja usando o sistema operacional Linux (Ubuntu por exemplo), deve-se mandar executar em terminal o arquivo ProjetoRedes_Alysson_Felipe.sh, que se encontra na raiz do diretório. No terminal que se abrirá, realize a escolha no menu entre:
	
	1 se deseja alterar as configurações; 
	2 usar as configurações padrão;
	3 para encerrar o programa; 
	e dê ENTER.
	
	Se escolher a opção 1, alterar as configurações, irá alterar o time-out dos roteadores, bem como o tempo para envio de cada pacote de dados.
	Após alterar as configurações você escolherá o roteador que deseja criar e iniciará o programa de roteamento. 
	
	Escolhendo a opção 2, irá escolher um roteador e usará as configurações definidas como padrão no programa.
	 
	Para matar um nó, dê Ctrl+c ou feche a janela do terminal.
